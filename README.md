# Stock Analysis Dashboard

This project is a Streamlit web app that analyzes stock performance, risk, and portfolio optimization.

## Features
- Multiple stock selection (3–10 tickers)
- Return and risk analysis
- Correlation heatmap
- Portfolio optimization (Equal Weight, GMV, Tangency)
- Efficient Frontier and CAL
- Custom portfolio builder

## How to Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
